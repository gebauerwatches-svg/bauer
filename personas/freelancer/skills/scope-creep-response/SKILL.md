# Handling scope creep

When the user asks for help responding to a scope-expansion request:

## Step 1 — Understand

Ask in one short question (skip if obvious):
- **What did the client originally agree to?** One sentence.
- **What are they asking for now?** One sentence.
- **Long-term client or one-off?**

## Step 2 — Identify the situation

Three common scenarios:
- **Small ask, actually within scope** — fine, just confirms they didn't read the SOW. Reassure briefly and proceed.
- **Small ask, outside scope** — could be a goodwill gesture OR a precedent-setter. The user decides.
- **Large ask, outside scope** — needs a paid change order.

## Step 3 — Draft the response

Always lead with appreciation (so it doesn't feel transactional) → state the scope reality → give a path forward.

**Template — small but outside scope:**

```
Hey [Name],

Glad you're thinking ahead on this! Quick note that [the new ask] is a bit outside what we scoped originally, but I can [option A: fold it in for free as a one-time goodwill thing / option B: handle it for a small additional $X].

Want me to go ahead with [option] — or would you rather we revisit the scope together?
```

**Template — larger, needs a change order:**

```
Hi [Name],

Thanks for sharing this — sounds like the project is evolving in a useful direction.

[The new ask] is outside our original scope, so I'd want to handle it as a separate piece of work. My estimate: [N hours] / $[total]. Happy to put together a quick add-on SOW if you want to move forward, or we can keep the original project on track and revisit this after.

Either way works. What sounds best?
```

## Rules

- **Always lead with warmth.** This is a relationship, not a contract dispute.
- **Always offer a clear path forward.** Don't say "no" and leave them hanging.
- **Default to billing for scope expansion.** Doing "free goodwill work" compounds — clients learn that scope is negotiable.
- **Make the ask explicit.** "Want me to write up the change order?" forces a yes/no.

## Special cases

- **If the user is uncomfortable charging:** offer a mid-tier option — *"happy to handle this one as a goodwill gesture, but flagging that I'd typically scope this separately."*
- **If the client has been a strong long-term relationship:** the goodwill option is more reasonable. Note this to the user.
- **If the client is consistently scope-creeping:** flag the pattern to the user. Suggest a "monthly retainer" framing as a sustainable response.

End with: "Want this firmer, warmer, or more specific about the dollars?"
