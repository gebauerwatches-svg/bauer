Beyond setup

Build agents in plain English.

Every Bauer install includes a build-agent skill. Just say “help me build an agent that…” and Bauer walks you through it.

  • Define what it does. Plain-English purpose, trigger, output. One sentence each.
  • Set careful permissions. Should it read your Drive but never delete? Read your inbox but not reply? Tell Bauer in everyday language.
  • Pick your model. Closed-source (Claude Pro, $20/mo) for the easiest path. Or open-source local (Ollama + Llama or Mistral) for free and private.
  • Set hard guardrails. Rules the agent will never break — like “never send anything without showing me first.”
  • Bauer writes the agent file. You get a trigger phrase, an editable file, and one example prompt to try right away.

Or: open-source local AI

Run an AI on your own computer instead.

Same Bauer experience — different AI underneath. Free to run, more privacy, no subscription. About 5 minutes to install (downloads ~4 GB). Slower and less capable than Claude on complex tasks, but great for privacy-sensitive work.

🍎 Mac / Linux

curl -sSL https://bauerai.vercel.app/install-local.sh | bash

🪟 Windows (PowerShell)

iwr https://bauerai.vercel.app/install-local.ps1 | iex

The local installer auto-installs Ollama, downloads an open-source model (qwen2.5-coder:7b), installs Claude Code, and sets up the bridge between them. After install, run ccr code to start chatting with your local AI.

What you’ll need

Two paths to the AI itself — pick one:

  • Closed-source (recommended for most people) — A Claude Pro subscription ($20/mo). Predictable, fastest, easiest. Bauer’s installer sets it up. The free claude.ai tier alone won’t be enough.
  • Open-source (free) — A local model via Ollama. Runs entirely on your computer. More privacy, no monthly cost, slower on complex tasks. Now fully automated — use the open-source installer above.

Common questions

What does this actually install?

Bauer drops a curated config file (a CLAUDE.md) and 4 role-specific skills into your computer’s AI settings. The config tunes the AI’s tone and defaults for your role. Three of the skills cover the 80% of tasks you’ll actually do, plus a build-agent skill that walks you through designing custom agents in plain English.

How is this different from setting up AI yourself?

By default, when you install the AI tool, it comes with no config and no skills — you’re starting with a blank assistant that doesn’t know who you are or how you work. Bauer drops in the config you’d otherwise spend hours figuring out, and gives you a guided agent builder so you can extend it without learning the underlying file format.

Can I use open-source models instead of Claude?

Yes — there’s a separate one-command installer (see above) that auto-installs Ollama, downloads a local model, installs Claude Code, and sets up the bridge between them. Same persona/skills experience, just slower and free. The build-agent skill also asks about model choice when you design new agents.

What AI does Bauer use by default?

Right now, Anthropic’s Claude (via the terminal version of Claude Code, under the hood). We may add more options based on what people actually want.

I already have a CLAUDE.md. Will Bauer overwrite it?

No. We back up your existing CLAUDE.md to CLAUDE.md.before-bauer.[timestamp] before writing the new one. Same for any skill folders that collide. Nothing is lost.

How do I uninstall?

Delete ~/.claude/CLAUDE.md and the relevant ~/.claude/skills/ subfolders (or restore your backups). The underlying AI tool stays installed; you can always run it with no config and it works fine.

Is my data safe?

Same as the default AI install — the AI only reads files you explicitly ask it to. Bauer doesn’t add any data collection. The configs live entirely on your computer. With the open-source path, nothing leaves your machine at all.

How much does it cost?

Bauer itself is free and open source. To use the AI, either pay for Claude Pro ($20/mo — recommended) or set up an open-source local model (free, more setup). The free claude.ai tier alone won’t be enough.
