Pick your computer + run the command

The local installer auto-installs Ollama (free software that runs AI models on your computer), downloads an open-source model, installs Claude Code, and sets up the bridge between them. After install, run ccr code to start chatting with your local AI.

🍎 Mac / Linux

curl -sSL https://bauerai.vercel.app/install-local.sh | bash

🪟 Windows (PowerShell)

iwr https://bauerai.vercel.app/install-local.ps1 | iex

What you’ll need

  • A Mac or Windows computer with at least 8 GB RAM (16 GB+ recommended)
  • ~5 GB of free disk space (for the AI model)
  • About 5–10 minutes (most of which is the model download)
  • No subscription needed — everything runs locally and is free

Local open-source models are slower and less capable than Claude on complex tasks, but great for privacy-sensitive work and zero ongoing cost. For complex agent work, switch back to the Claude path.
